# meme_conditions

Working repository for a corpus study of community-documented use conditions
of internet meme templates, built on the SemioMeme/KYM data.

- CLAUDE.md — project brief and ground rules (read first)
- GATE_SPEC.md — day-1 viability checks
- config.py — local data paths and parameters

No data is stored in this repository. The released dataset lives at
github.com/vemchance/semiomeme (Zenodo: 10.5281/zenodo.17826799).
